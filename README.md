# Ai-Traffic-Violation-Detection- 
# Summary-
This project uses computer vision and Ai methods to automatically detected traffic rules violations from Camera footage the system identifies vehicles, tracks their movements and flags violations such as overspeeding, signal jumping,and lane discipline issues. the goal is to demonstrate how AI can improve road safety, support authorities, and reduce human error in traffic monitoring.the project includes data preprocessing a simple detection model and a clear explanation of how AI methods are applied.
